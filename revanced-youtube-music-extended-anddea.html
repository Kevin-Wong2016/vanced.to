ReVanced YouTube Music Extended (anddea)

ReVanced YouTube Music Extended (anddea) (RVX Music) is an extended and customized version of the YouTube Music Extended (anddea) app. It is designed to provide additional features, functionalities, and customization options beyond what the official YouTube Music Extended (anddea) app offers.

ReVanced YouTube Music Extended (anddea) Features

  • Pure black theme: Applies a pure black theme to some components.
  • Background music: Enables playing music in the background.
  • "Always High" audio quality: Sets the audio quality to "Always High" when you first install the app.
  • Android Auto Support: Enables YouTube Music to work with Android Auto by spoofing the YouTube Music certificate.
  • Customizable app opening page: Adds an option to set which page the app opens in instead of the homepage.
  • Customizable launcher icon: Changes the YouTube Music app icon to the icon specified in options.json.
  • Customizable YouTube Music app name: Renames the YouTube Music app to the name specified in options.json.
  • Customizable package name: Changes the package name for the non-root build of YouTube and YouTube Music to the name specified in options.json.
  • Customizable playback speeds: Adds an option to customize available playback speeds.
  • Disable captions: Adds an option to disable captions from being automatically enabled.
  • Remove dark overlay: Removes the dark overlay when comment, share, save to playlist, and flyout panels are open.
  • Set navigation bar color: Adds an option to set the navigation bar color to black.
  • Match miniplayer color: Adds an option to match the color of the miniplayer to the fullscreen player. Deprecated on YT Music 6.34.51+.
  • Enable compact flyout menu: Adds an option to enable the compact flyout menu on phones.
  • Custom filter: Adds a custom filter which can be used to hide layout components.
  • Debug logging: Adds an option to enable debug logging.
  • Keep miniplayer minimized: Adds an option to keep the miniplayer minimized even when another track is played.
  • Enable landscape mode: Adds an option to enable landscape mode when rotating the screen on phones.
  • Playback in miniplayer for Kids music: Enables playback in miniplayer for Kids music.
  • Old style player background: Adds an option to return the player background to the old style. Deprecated on YT Music 6.34.51+.
