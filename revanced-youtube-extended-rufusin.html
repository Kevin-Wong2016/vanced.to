ReVanced YouTube Extended Features

  • Old Style Splash Animation: Adds the old style splash animation to the app loading screen.
  • Replace Video Thumbnails: Adds options to replace video thumbnails using the DeArrow API or image captures from the video.
  • Ambient Mode Bypass: Adds an option to bypass the restrictions of ambient mode or disable it completely.
  • Current Video Quality/ Playback Speed Display: Adds an option to display the current video quality or playback speed in brackets next to the current time.
  • Text Toggles in Additional Settings: Adds an option to use text toggles instead of switch toggles within the additional settings menu.
  • Custom App Opening Page: Adds an option to set which page the app opens in instead of the homepage.
  • Custom App Heading: Applies a custom heading in the top left corner within the app.
  • Custom Launcher Icon: Changes the YouTube launcher icon to the icon specified in options.json.
  • Custom App Name: Renames the YouTube app to the name specified in options.json.
  • 'Double-tap to Seek' Value: Adds a 'double-tap to seek' value to improve video navigation.
  • Custom Package Name for non-root Build: Changes the package name for the non-root build of YouTube and YouTube Music to the name specified in options.json.
  • Customizable Playback Speeds: Adds options to customize available playback speeds.
  • Video Player Opacity: Adds an option to change the opacity of the video player background when player controls are visible.
  • Seekbar Colors Customization: Adds an option to customize seekbar colors in video players and video thumbnails.
  • Default Playback Speed: Adds an option to set the default playback speed.
  • Default Video Quality: Adds an option to set default video quality.
  • Disable HDR Video: Adds options to disable HDR video.
  • Disable CronetEngine's QUIC Protocol: Adds an option to disable CronetEngine's QUIC protocol.
  • Disable Automatic Captions: Adds an option to disable captions from being automatically enabled.
  • Disable Haptic Feedback: Adds an option to disable haptic feedback when swiping the video player.
  • Disable Landscape Mode: Adds an option to disable landscape mode when entering fullscreen.
  • Disable PIP Notification: Disables the picture-in-picture (PIP) notification when you first launch PIP mode.

What's the difference between ReVanced YouTube and ReVanced YouTube Extended?

ReVanced YouTube and ReVanced YouTube Extended are both modded versions of the official YouTube app that add additional features and remove ads. However, there are some key differences between the two versions.

YouTube ReVanced

  • Basic Features: Includes enhancements like ad-blocking and background playback.
  • User Interface: May have a different user interface compared to ReVanced Extended.
  • Customization: Offers basic customization options.
  • Development: Regular updates based on the development team's schedule.

YouTube ReVanced Extended

  • Extended Features: Offers all features of ReVanced plus additional ones like video downloading.
  • Enhanced Customization: More extensive customization options including themes, layouts, and feature toggling.
  • Continual Development: Committed to ongoing development and updates for performance and stability.
  • Reliable Ad-Blocking: Praised for more effective ad-blocking capabilities.
